# Docs

## Extra Variable
```bash
{username} print username.
{userfirstname} print user first name.
{userlastname} print user last name.
{userid} print user id.
{userfullname} print user full name.
{userlocation} print user location eg indonesia.
{chattitle} print title chat.
{chatusername} print username  chat.
{chattype} print chat type eg private.
{chatid} print chat id.
{suasana} print suasana eg siang.
{jam} print jam eg 1.
{menit} print menit eg 15.
{detik} print detik eg 60.
{hari} print hari eg senin.
{bulan} print bulan eg januari.
{nobulan} print no bulan eg 1.
{tahun} print tahun eg 2021.
{daymonthyear} print date eg senin-agustus-2021.
{monthdayyear} print date eg agustus-senin-2021.
{ddmmyear} print date eg 20-7-2021.
{mmddyear} print date eg 7-25-2021.
```
